Problem: "git: command not found"
Solution: Install Git from https://git-scm.com/download/win

Problem: "Permission denied (publickey)"
Solution: Use HTTPS (not SSH) and create Personal Access Token

Problem: "Updates were rejected"
Solution: git pull origin main, then git push

Problem: Website shows 404
Solution: 
  - Verify file is named "index.html" (not kamfire-pixel-site.html)
  - Check GitHub Pages settings
  - Wait 2-3 minutes after pushing
  - Clear browser cache

Problem: Custom domain not working
Solution:
  - Verify CNAME file exists in repository root
  - Check DNS settings in GoDaddy
  - Wait for DNS propagation (up to 48 hours)
  - Make sure DNS records are exactly as shown above
